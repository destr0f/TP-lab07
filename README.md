# TP-lab04

Лабораторная работа №4 по ТиМП.

## Описание

Проект содержит простую функцию `print`, объявленную в `include/print.hpp` и реализованную в `sources/print.cpp`.

Сборка проекта выполняется через `CMakeLists.txt`. Проверка сборки настроена через GitHub Actions.

## Структура проекта

- `include/` — заголовочные файлы
- `sources/` — исходные файлы
- `examples/` — примеры использования
- `CMakeLists.txt` — файл сборки CMake
- `.github/workflows/cmake.yml` — workflow GitHub Actions
