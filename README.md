# TP-lab03

Лабораторная работа №3 по ТиМП.

## Описание

Проект содержит простую функцию `print`, объявленную в `include/print.hpp` и реализованную в `sources/print.cpp`, а также файл `CMakeLists.txt` для сборки проекта через CMake.

## Структура проекта

- `include/` — заголовочные файлы
- `sources/` — исходные файлы
- `examples/` — примеры использования
- `CMakeLists.txt` — файл сборки CMake
